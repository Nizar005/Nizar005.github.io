Tentang Situs

Semua ini dimulai ketika saya memposting karya blender saya di instragram. saya pikir "akan sulit untuk memposting hanya ke instagram, karena hanya ada beberapa rasio gambar untuk dipilih, saya mungkin berpikir dua kali jika saya ingin membuat proyek yang memiliki berbagai rasio gambar" dari itu saya memiliki ide untuk membangun situs web , lumayan untuk pengalaman. pertama kali saya membuat web di wordpress yang dihosting oleh ProFreeHost. meskipun gratis dan memiliki ruang penyimpanan tidak terbatas, hostingnya buruk sekali. beberapa kali saya tidak bisa masuk ke situs web saya. jadi saya pindah ke situs resmi Wordpress. tetapi versi gratisnya memiliki batasan ukuran penyimpanan. jadi saya memutuskan untuk belajar css html dan mungkin javascript. selain bisa membuat website lebih fleksibel, ini kelihatan keren dimata orang awam.
